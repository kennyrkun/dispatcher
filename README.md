# Requirements
- ollama and llama3.2
- ffmpeg
- ffplay
- openai whisper
- pyaudio
- piper-tts
  - If using pip to install piper, follow these steps:
    - install piper-phonemize-cross
    - install piper-tts
    - install onnxruntime
  - voice data is required for piper. The program looks for voice data in `./voices/`
